# 🤖 Hands-on Reinforcement Learning

This repository is designed to help you understand reinforcement learning from the ground up. Instead of focusing only on theory, it emphasizes implementation, experimentation, and analysis of results.

---

## 📚 Topics Covered

- Fundamentals of Reinforcement Learning
- Markov Decision Processes (MDPs)
- Exploration vs Exploitation
- Value-Based Methods
- Policy-Based Methods
- Model-Free Reinforcement Learning
- Deep Reinforcement Learning (introductory)

---

## 🛠️ Algorithms Implemented

🔹 Dynamic Programming (Model-Based)
- Policy Iteration: Evaluating and improving policies to reach optimality.
- Value Iteration: Iteratively computing the optimal state-value function.
